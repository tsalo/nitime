Following the latest source¶

These are the instructions if you just want to follow the latest nitime source, but you don’t need to do any development for now.

The steps are:

  • Install git
  • get local copy of the git repository from github
  • update local copy from time to time

Get the local copy of the code¶

From the command line:

git clone git://github.com/nipy/nitime.git

You now have a copy of the code tree in the new nitime directory.

Updating the code¶

From time to time you may want to pull down the latest code. Do this with:

cd nitime
git pull

The tree in nitime will now have the latest changes from the initial repository.
